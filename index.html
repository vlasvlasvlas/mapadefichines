Qué es esto?

  •  Sobre el proyecto
  •  Disclaimer

Este proyecto colaborativo arrancó con la idea de tener un lugar mapeado donde encontrar lugares donde haya fichines para jugar y compartir. Podés colaborar desde el repo github disponible en MapaDeFichineso contactandonos vía instagram o buscando en cybercirujas.

Podés sumar tus puntos contactandonos por el proyecto en github, escribiendonos al instagram o a través del grupo de CyberCirujas

El código se basa en un fork de plantilla html+js del proyecto Bootleaf, de licencia MIT, y disponible en GitHub Bootleaf.

"Los datos proporcionados en este sitio son únicamente para fines informativos y de planificación. No se implica ni se pretende ninguna garantía de exactitud o completitud. Toda la información en este mapa está sujeta a variaciones y correcciones.
